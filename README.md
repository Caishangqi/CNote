## 介绍 Introduction
基于 C++ 14 的知识合集，主要面向游戏开发所使用的知识点。相比于其他文档，该文档提供C++特性的使用说明和解释包括为什么以及什么时候使用，更偏向底层原理。对于未接触编程学者而言不是一篇十分适用的教程。
